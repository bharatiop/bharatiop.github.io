<div class="section">
  <p style="font-size:14.5px;color:var(--text-muted);line-height:1.85;max-width:680px;margin-bottom:24px;">
    This website is built with <a href="https://jekyllrb.com" target="_blank" rel="noopener" style="color:var(--accent-mid);">Jekyll</a>
    and hosted on <a href="https://pages.github.com" target="_blank" rel="noopener" style="color:var(--accent-mid);">GitHub Pages</a>.
    The design uses a custom CSS system (no framework dependency) with a clean academic aesthetic — Inter and DM Serif Display typefaces,
    a muted navy-and-white palette, and card-based layouts for research, publications, and team content.
  </p>
  <p style="font-size:14.5px;color:var(--text-muted);line-height:1.85;max-width:680px;margin-bottom:24px;">
    The homepage includes a live clock and calendar widget showing upcoming field conferences, and an AI-curated panel
    summarizing recent developments in neutron-star and gravitational-wave astrophysics.
  </p>
  <p style="font-size:14.5px;color:var(--text-muted);line-height:1.85;max-width:680px;margin-bottom:32px;">
    Students in the group are welcome to edit and contribute to the site. The source is available on
    <a href="https://github.com/bharatiop/bharatiop.github.io" target="_blank" rel="noopener" style="color:var(--accent-mid);">GitHub ↗</a> —
    pull requests for typo fixes, new publications, or news updates are welcome.
  </p>
